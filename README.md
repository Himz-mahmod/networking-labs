# Inter-VLAN Routing Lab (Router-on-a-Stick)

## Overview
This project demonstrates VLAN segmentation and inter-VLAN routing using Cisco Packet Tracer.

## Network Design
- VLAN 10: HR (192.168.10.0/24)
- VLAN 20: Accounts (192.168.20.0/24)
- Router-on-a-Stick configuration
- IEEE 802.1Q trunking

## Key Features
- VLAN creation and port assignment
- Trunk port configuration
- Sub-interface configuration on router
- Inter-VLAN communication verified using ICMP (ping)

## Technologies Used
- Cisco 2960 Switch
- Cisco ISR Router
- 802.1Q encapsulation
